## Development Workflow

This project uses [OpenSpec](https://openspec.dev/) for spec-driven development. OpenSpec provides a structured workflow for planning and implementing features using AI assistants.

### Quick Start with OpenSpec

To propose a new change:
```bash
/opsx:new <change-name>
```

For guided onboarding:
```bash
/opsx:onboard
```

For complete workflow documentation, see [openspec/README.md](openspec/README.md).
